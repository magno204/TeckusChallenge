# Layout Module

Esta carpeta contiene los componentes de layout de la aplicación (header, sidebar, footer, etc.).

## Componentes:
- **header/**: Componente del encabezado
- **sidebar/**: Componente de la barra lateral
- **footer/**: Componente del pie de página

## Uso:
Estos componentes se pueden usar en el componente principal de la aplicación o en componentes de layout específicos.
