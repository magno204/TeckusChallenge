# Shared Module

Esta carpeta contiene recursos compartidos que pueden ser utilizados en múltiples features.

## Estructura:
- **components/**: Componentes reutilizables
- **pipes/**: Pipes personalizados
- **directives/**: Directivas personalizadas

## Reglas:
- No debe contener lógica de negocio específica
- Debe ser completamente independiente de features
- Puede ser importado por cualquier módulo
